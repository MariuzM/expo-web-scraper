<h1 align="center">Project - Expo Web Scraper</h1>

### Requirements

- Node.js 16.20.0 LTS https://nodejs.org/en

- PNPM https://pnpm.io

### Install and start App

```
pnpm install
pnpm dev
```

### Build and start App with Safari Exstention

```
pnpm build
```

## Features

1. Tab navigation
2. Scrape time and display in two components
3. Safari Exstention
